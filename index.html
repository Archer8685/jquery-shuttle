How to Use It

javascript

$(".shuttle_select").shuttle();

html

just add shuttle_select in class and set multiple to your select.

<select class="shuttle_select"></select>

API

Options

  • filter
    open filter option funcation.
  • srcTitle
    source select title.
  • dstTitle
    target select title.
  • dstValue
    target select default option.

Events

  • reset
    reset shuttle value.
  • removeAll
    remove all option.
  • disable
    disable shuttle function.
  • enable
    enable shuttle function.
